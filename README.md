# Validador de Contraseñas 🔐
## Descripción

Aplicación móvil desarrollada con React Native y Expo que permite validar si una contraseña cumple con criterios básicos de seguridad como longitud mínima, uso de mayúsculas y números.

Instalación y ejecución

1. Clonar el repositorio:
git clone https://github.com/willy460/validador-contrasenas.git

2. Instalar dependencias:
npm install

3. Ejecutar la aplicación:
npx expo start

Luego se puede abrir la aplicación en:

- Expo Go (dispositivo móvil)
- Emulador de Android
- Simulador de iOS
  
Funcionalidades
- Ingreso de contraseña por parte del usuario.
- Validación de requisitos de seguridad.
 -Mensaje indicando si la contraseña es segura o no.
